# node-express-architecture-example

## Description

Repository contains nodejs-api for a e-commerce source code.

## Developing

```
$ run nodejs-api.sql script into your database management system
$ cp .env.example .env
$ npm install
$ npm run dev:start
$ navigate to `http://localhost:3000/api/v1/docs`
```
